Di situs ini saya akan menjelaskan beberapa class yang tersedia pada framework 'w3.css' yang berfungsi untuk memberikan animasi kepada sebuah elemen. Beberapa contoh class tersebut diantaranya:

  • Top
  • Bottom
  • Left
  • Right
  • Zoom
  • Fading

w3-animate-top

w3-animate-top berfungsi untuk animasi suatu elemen dari atas ke bawah. Animasi tersebut akan terjadi sekali setiap kali page di-reload. Contoh w3-animate-top diaplikasikan pada gambar berikut:

w3-animate-bottom

w3-animate-bottom berfungsi untuk animasi suatu elemen dari bawah ke atas. Animasi tersebut akan terjadi sekali setiap kali page di-reload. Contoh w3-animate-bottom diaplikasikan pada gambar berikut:

w3-animate-left

w3-animate-left berfungsi untuk animasi suatu elemen dari kiri ke kanan. Animasi tersebut akan terjadi sekali setiap kali page di-reload. Contoh w3-animate-left diaplikasikan pada gambar berikut:

w3-animate-right

w3-animate-right berfungsi untuk animasi suatu elemen dari kanan ke kiri. Animasi tersebut akan terjadi sekali setiap kali page di-reload. Contoh w3-animate-right diaplikasikan pada gambar berikut:

w3-animate-opacity

w3-animate-opacity berfungsi untuk animasi suatu elemen dengan membuat ketebalan (opacity) elemen muncul dari 0% hingga 100%. Animasi tersebut akan terjadi sekali setiap kali page di-reload. Contoh w3-animate-opacity diaplikasikan pada gambar berikut:

w3-animate-fading

w3-animate-fading berfungsi untuk animasi suatu elemen dengan menghilangkan dan memunculkan elemen secara berulang kali. Animasi tersebut akan terjadi terus menerus selama halaman page masih jalan. Contoh w3-animate-fading diaplikasikan pada gambar berikut:

w3-animate-zoom

w3-animate-zoom berfungsi untuk animasi suatu elemen dengan membesarkan elemen dari ukuran 0% hingga ukuran 100%. Animasi tersebut akan terjadi sekali setiap kali page di-reload. Contoh w3-animate-zoom diaplikasikan pada gambar berikut:

w3-spin

w3-spin berfungsi untuk animasi suatu elemen dengan memutar elemen. w3-animate-spin akan berfungsi secara terus menerus selama situs page masih jalan. Contoh w3-spin diaplikasikan pada gambar berikut.
